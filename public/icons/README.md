# アプリアイコンの設定方法

このディレクトリには、PWAアプリアイコン画像を配置します。

## 必要なファイル

以下の2つのファイルを配置してください：

- `icon-192.png` - 192×192ピクセルのPNG画像
- `icon-512.png` - 512×512ピクセルのPNG画像

## 画像のリサイズ方法

提供されたブルームコネクトの画像を以下の方法でリサイズできます：

### オンラインツールを使用する方法

1. [Squoosh](https://squoosh.app/) にアクセス
2. 画像をアップロード
3. 右側のパネルで「Resize」を選択
4. 192×192 または 512×512 に設定
5. 形式を PNG に設定
6. ダウンロード

### Photoshop / GIMPを使用する方法

1. 画像を開く
2. イメージ → 画像サイズ → 192×192 または 512×512 に設定
3. ファイル → 書き出し → PNG形式で保存

### コマンドラインツールを使用する方法（ImageMagick）

```bash
# 192×192にリサイズ
convert bloom-connect-icon.png -resize 192x192 icon-192.png

# 512×512にリサイズ
convert bloom-connect-icon.png -resize 512x512 icon-512.png
```

## 配置後

画像を配置したら、以下を確認してください：

1. ファイル名が正しいこと（`icon-192.png` と `icon-512.png`）
2. このディレクトリ（`public/icons/`）に配置されていること
3. ブラウザのキャッシュをクリアして、アイコンが正しく表示されることを確認

## ファビコンの設定

アプリのファビコン（ブラウザタブのアイコン）を設定するには：

1. 同じ画像を 32×32 または 48×48 にリサイズ
2. `app/icon.png` として保存（Next.jsが自動的にfaviconとして使用します）

または、ICO形式を使用する場合：

1. 画像をICO形式に変換（[favicon.io](https://favicon.io/) などを使用）
2. `app/favicon.ico` として保存
